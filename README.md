# Employee & Department Management - MERN Stack

This project implements the Rails challenge equivalent using **React (frontend) + Node/Express (backend) + MongoDB (Mongoose)**.

## Structure
- backend/ - Express API with Mongoose models, controllers, routes and seed script
- frontend/ - React app (Create React App style) with pages and components

## Setup Backend
1. cd backend
2. npm install
3. create `.env` and set MONGO_URI
4. npm run seed
5. npm run dev

API base: http://localhost:5000/api

## Setup Frontend
1. cd frontend
2. npm install
3. create `.env` ans set REACT_API_KEY
4. npm start

Open http://localhost:3000
